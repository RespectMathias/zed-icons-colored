# Zed Icons Colored for VS Code

![Preview](https://raw.githubusercontent.com/RespectMathias/zed-icons-colored/refs/heads/main/images/comparison.webp)

This extension packages the colored Zed file icons as a Visual Studio Code file icon theme.

## Included

- broad file extension coverage tuned for VS Code
- dark and light theme overrides for icons that ship in both variants
- generic folder and file icons from the same asset set

## Source

The SVG assets live in `zed-icons-colored-theme/icons`. Original licensing notes remain in `NOTICE`. These assets are from [zed-icons-colored-theme](https://github.com/TheRedXD/zed-icons-colored-theme), which is licensed under the MIT License and [Zed editor](https://github.com/zed-industries/zed), which is licensed under the Apache License 2.0.

## Use

Install the extension folder, then choose `Colored Zed Icons` from the File Icon Theme picker in VS Code.
